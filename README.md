# Bharath Kumar & Moni Wedding Invitation

This project is a premium multi-page static wedding invitation designed for free GitHub Pages hosting.

## Pages

- `index.html` is the animated opening invitation cover.
- `welcome.html` is the welcome page with hero section and countdown.
- `celebration.html` contains venue and event details.
- `blessings.html` contains the RSVP and closing message.

## Edit Your Details

Open `script.js` and update:

- `weddingDate`
- `venueName`
- `venueAddress`
- `mapUrl`
- `rsvpUrl`
- `events`
- `openingMessage`
- `heroMessage`

## Publish On GitHub Pages

1. Create a GitHub repository, for example `bharath-kumar-moni`.
2. Upload all files from this folder to the repository.
3. Push everything to the `main` branch.
4. In GitHub, open `Settings` -> `Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Select branch `main` and folder `/root`.
7. Save.

Your website will be available at:

`https://YOUR-GITHUB-USERNAME.github.io/bharath-kumar-moni/`
